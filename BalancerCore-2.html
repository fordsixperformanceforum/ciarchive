Balancer Core Charge & Refund Policy

A core charge is a deposit for an exchange of the item purchased. Therefore all Classic Inline balancers include a $50-$75 core charge. All 1V balancers include a $50 core charge, while the 2V and 3V balancers include a $75 core charge. While the core charge is included in the price, it is refundable when you return the core. If payment was made using PayPal, or a credit card, the core must be returned within 60 days from the original purchase date to receive a full refund. If the core is returned after 60 days, a $10 fee will be deducted from the refund. If you purchased a 2V balancer, but return a 1V balancer, you will only receive a refund for the 1V core. You can also return more than one core, even though you only purchased one balancer, as Classic Inlines purchases any and all cores that are suitable for rebuilding. For additional cores or out right purchase, we pay $40 for 1V cores and $65 for 2V and 3v cores. We also pay $10 for bolt-on pulleys.

NOTE: A 2V balancer must be a true 2V, not a 1V with an extra pulley bolted on, to recieve the full core refund for a 2V balancer. If it is a 1V balancer with an additional bolt on pulley, the refund will be adjusted accordingly.

We recommended comparing the new balancer to your old one before you return the core, just to make sure it's the right one. You may use the box that the new one came in, to return the core, but it is not required. Shipping insurance is recommended, but that's up to you. However we highly recommend insurance if your returning several cores in one box.

The core is fully refundable if:

1. The core is rebuildable.  A rebuildable harmonic balancer core is free from damage other than missing, or cracked rubber.  If the only problem with the damper is that the rubber has let loose, then the core is usually rebuildable.  Partial credit may be available in some cases where there are some useable core components. 

    Common problems which would make the core unsuitable for rebuilding are:
            a. cracked or bent hubs
            b. chipped, broken or missing outer rings
            c. worn or cracked crank keyways

    Classic Inlines or its rebuilder, Damper Docs, have the final judge on the condition.

2. The core is the same as the one purchased.  The core being returned to Classic Inlines must be of the same year and engine size of the balancer that was purchased to receive the full refund. However you can return a different core and still get a partial refund. For example: if you purchase a 2V balancer, but return a 1V balancer, you will only receive a refund for the 1V core. However since we purchase any and all cores that are in good condition and suitable for rebuilding, you can also return more than one core if you like.

3. The core return must include the proper information. All cores must include the name of the original purchaser, your address and phone number (including the area code), and the original method of payment. Credit will be issued in the same matter in which you originally paid for the balancer, and will only be refunded to the original purchaser. Cores returned or sent in without the proper paperwork are not eligible for a refund or purchase.

4. The core return must be within 60 days. If payment was made using PayPal, the core must be returned within 60 days from the original date of purchase to receive a full refund. If the core is not received within sixty days from the original date of purchase, a 10% fee will be deducted from the refund. No exceptions.
