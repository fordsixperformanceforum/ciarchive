C/R Charts - 250ci with a Zero Deck Height

The charts below were calculated using Virtual Engine 2000's Cam Utility Program. The charts list both Static and Dynamic C/R's for a 250ci engine with a zero deck height, using varied chamber sizes and valve timing (intake close). Calculations were computed using a Felpro head gasket which has a .050 compressed thickness and a 3.81 gasket bore. The columns to the left use dished pistons (6.5cc), the columns to the right use flat top pistons. You can use the links below for the bore size, or just scroll down.
